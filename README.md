CRPS is the Coldreader Persistent Storage system.
Not much here yet.

## Install

Via Composer

``` bash
$ composer require imonroe/crps
```

## Usage

``` php
$cprs = new imonroe\crps();
echo $cprs->echoPhrase('Hello, world');
```
